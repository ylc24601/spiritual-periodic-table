# 卡片設計與輸出說明

## 目標
整理現有 `spiritual_element_cards_v2_1.html` 的卡片設計規範，並提供可列印或轉成簡報的輸出方案。

## 現有設計重點
- 四種屬靈類型配色：
  - 行動型 `--action-main` #FF7F6B
  - 支撐型 `--struct-main` #F0A840
  - 連結型 `--rel-main` #4EC8B0
  - 穩定型 `--stab-main` #8AAEE0
- 卡片圓角：14px
- 正反面翻轉動畫採 CSS 3D transform
- 卡片正面內容：原子序、中文名、符號、英文名、屬靈稱號、適合對象
- 卡片背面內容：科學特性、屬靈特質、弱點提醒、經文、聖經人物對應

## 卡片輸出建議
### 1. HTML 轉列印 PDF
- 使用瀏覽器直接開啟 `spiritual_element_cards_v2_1.html`。
- 若需列印實體卡片，建議先把卡片改成單面版或者雙面整頁排版。
- 透過 Chrome / Safari 的「列印」功能，選擇「儲存為 PDF」或直接列印。
- 建議印刷尺寸：每張卡片約 70x100 mm 或 80x120 mm。

### 2. 雙面卡片設計
- 正面為原子類比，背面為屬靈說明。
- 如果要製作可翻的實體卡，可以先列印兩張同尺寸紙張，再手工貼合。
- 若要直接輸出為兩面 PDF，可把 HTML 調整為「正面頁」與「背面頁」，再使用雙面列印。

### 3. PPT / 簡報圖板輸出
- 若希望轉成簡報素材，可截圖 `spiritual_element_cards_v2_1.html` 卡片區塊：
  - 每張卡片單獨截圖或批量截圖
  - 置入 PPT 模版，調整成 16:9 或 4:3 版面
- 也可請資訊同工協助：
  - 用現有 HTML 把卡片截圖成 PNG
  - 將卡片資料生成 PPT 內容

### 4. PDF 列印注意事項
- 頁面邊距：建議 10-12mm
- 模式：彩色列印會呈現最完整的色彩，而黑白列印則容易失去類型對應感。
- 圖檔解析度：截圖建議至少 150-200 dpi，避免放大失真。

## 內容確認項目
請在輸出前確認以下內容是否正確：
- 元素標題是否貼近屬靈角色
- 目標對象描述是否清楚
- 科學特性與屬靈特質是否一致
- 弱點提醒是否帶恩典與提醒
- 聖經經文與人物對應是否恰當
- 化合物比喻是否有清楚的神學連結

## 視覺一致性建議
- 文字排版：標題使用醒目字體，內文保持簡潔易讀。
- 文字層次：
  - 正面大標題：元素符號
  - 次標：元素中文名、英文名
  - 正文：屬靈稱號與推薦對象
- 背面重點區塊：
  - 科學特性
  - 屬靈特質
  - 留意弱點
  - 經文引用

## 建議後續操作
1. 若要直接做成實體卡，可先列印測試版 2-3 張，確認尺寸與排版。
2. 若要做 PPT 講義，可請同工把卡片內容整理成「一張卡片一個投影片」或「每二張卡片一頁」。
3. 若要做 PDF 手冊，可將卡片內容排成「左右對照」格式：左邊元素卡正面，右邊屬靈說明背面。
