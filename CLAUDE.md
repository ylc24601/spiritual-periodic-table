# 上帝的化學配方：屬靈元素週期表

## 專案說明
這是一個教會社青與長輩聯合聚會的互動圖卡工具。
帶領者：生化教授／社青輔導，活動時間 60-90 分鐘。
核心經文：哥林多前書 12:14, 18

## 技術架構
- 純前端，單頁 HTML（無框架）
- 字型：Varela Round（符號）、Noto Serif TC（中文標題）、Noto Sans TC（內文）
- 卡片翻面動畫：CSS 3D transform

## 設計規範
- 四種屬靈類型配色：
  - 行動型 --action-main: #FF7F6B
  - 支撐型 --struct-main: #F0A840（化合物雙關語顏色）
  - 連結型 --rel-main:    #4EC8B0
  - 穩定型 --stab-main:   #8AAEE0
- 暗色系統（Header、化合物黑板區）：
  - --board:      #1c2340（深海軍藍）
  - --board-text: #f0ede4（暖白）
- 頁面背景：#f5f3ef + 1px dot grid（CSS radial-gradient）
- 卡片圓角：14px
- 元素符號字型：Varela Round, 粗體

## 頁面結構邏輯
- Header 和化合物區塊（.board-section）同為深海軍藍，形成首尾呼應的「實驗室」感
- 中間卡片格（.grid）是溫暖的「團契空間」
- 化合物公式（.cformula）大字 44px 白色，雙關語（.cpun）用琥珀色 --struct-main

## 元素卡結構（正面）
原子序（左上）／中文名（右上）／符號（大，中央）／英文名（符號下）／屬靈稱號／適合對象

## 活動流程（60–90 分鐘）
1. 兩個團契混合，重新分小組
2. 每人從 16 張元素卡中，挑 1–2 張最符合自己的（提醒翻面看說明）
3. 小組員輪流分享選了哪張、為什麼選它（**圖卡分享時間由主帶統一控制，時間到主帶喊停收齊全場**）
4. 主帶先說明配對原則，並示範 1–3 組化合物，再讓各組配對
5. 各組把自己挑的元素卡浮貼到「共用大海報」上對應的化合物／混合物團契（紙膠帶黏卡片上緣、寫名字；卡片可掀起看背面說明）。**不需把全部化合物與混合物都湊在同一張海報上，只把有配對到的貼上即可；怎麼貼由各組自己決定——代表不同弟兄姊妹團契時會有不一樣的化學反應**
6. 全體一起看完成的海報，由同工帶大家欣賞各化合物團契的組合，邀幾位分享為何貼在那一格

## 重要慣例
- 修改元素內容請同步更新 JS 的 elements 陣列（script.js）
- 修改化合物請同步更新 JS 的 compounds 陣列（script.js）
- 不要動現有 CSS 變數名稱，可新增 --board、--board-text 等
- 化合物區塊在 HTML 用 .board-section 包裝，才能套用黑板暗色主題
- 所有中文使用繁體
